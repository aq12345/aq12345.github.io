分享一些超好用的DNS地址

发表于 2017-09-06 | 阅读次数:

##公司的网络最近经常抽风，尤其是在连接国外网络的时候。有一种原因是运营商DNS劫持了。下面我提供一些DNS来修改当前的DNS以应对运营商劫持的情况

  • 第一步 打开系统偏好设置进入网络，然后点击高级这个选项

  • 第二步，选择DNS选项，点击左下角＋号建，添加新的可用DNS。然后保存

  • 第三步，下面的DNS任选一个或者两个填进去，好吧，其实我就是想保存一下这几个DNS留作以后备用，顺便写个小步骤教程。－。－
    我这里有几个备用可以连接App Store的dns：
    OpenDNS:208.67.222.222和208.67.220.220
    GoogleDNS:8.8.8.8和8.8.4.4
    NortonDNS:198.153.192.1和198.153.194.1
    comodoDNS:156.154.70.221和56.154.71.22
    114DNS:114.114.114.114
    其中Norton和Comodo的都加入了安全功能，对带毒和挂马、有恶意代码的网站会进行拦截
    强烈推荐opendns，虽然谷歌的8888是秒进，但是下载速度并不快，这个很快！
